How this was built

methodology & limits

Ratings. Each team carries a World Football Elo rating (eloratings.net scale). The twelve strongest are exact values as of 10 June 2026; the rest are same-scale recent snapshots. Hosts USA, Canada and Mexico get a +65 Elo home bump.

From Elo to goals. The rating gap sets each side's expected goals via λ = 1.33·e^(0.88·ΔElo/400), calibrated so the model's win probability matches Elo win expectancy. Scorelines come from independent Poisson draws; the printed score is the single most-likely grid cell.

Group tables are ranked by expected points (3·P(win)+1·P(draw) over three games), with Elo breaking ties as a proxy for head-to-head and goal difference.

Knockouts use the real bracket wiring, including FIFA's Annex C allocation of the eight best third-placed teams. A drawn knockout is settled by Elo win expectancy (the extra-time / penalties proxy). The bracket shown sends the favourite through each round.

Why a separate model. This replaced an earlier attempt on a synthetic player dataset whose match results carried no learnable signal — every team-strength model there scored worse than a base-rate baseline. Real Elo is what makes these predictions meaningful.

Known limits. Pure Elo slightly over-concentrates the favourites over eight knockout games; lower-tier ratings are estimates; no home advantage beyond the host bump; draws use plain Poisson (no Dixon-Coles low-score correction). Relative ordering is sound; exact percentages are model-implied.

Generated from a reproducible Python model (Elo → Poisson → Monte Carlo). Predictions reflect pre-tournament ratings and do not update for results. Not betting advice.
